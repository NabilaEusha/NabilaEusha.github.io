Embedded Systems

ESP32-Based Mobile Charging Station

Developed a coin-operated and RFID-enabled mobile charging station using the ESP32 microcontroller. The system supports four independent relay-controlled charging ports with real-time status display on an LCD, automatic time-based cutoff, and secure public usage. The logic was first validated using Wokwi simulation before full hardware implementation.
